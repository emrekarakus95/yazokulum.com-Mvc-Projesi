ASP.NET uygulamanız

Tebrikler! Bir proje oluşturdunuz

Bu uygulama şunları içerir:

  • Ana Sayfa, Hakkında ve İletişim arasında gezinmeyi sağlayan örnek sayfalar
  • Bootstrap kullanarak tema oluşturma
  • Kimlik Doğrulaması, seçili olduğunda kayıt ve oturum açma yöntemlerini gösterir
  • ASP.NET özellikleri NuGet kullanılarak yönetilir

Uygulamayı özelleştir

  • ASP.NET MVC kullanmaya başlayın
  • Sitenin temasını değiştirin
  • NuGet kullanarak daha fazla kitaplık ekleyin
  • Kimlik doğrulamayı yapılandırın
  • Web sitesi kullanıcılarıyla ilgili bilgileri özelleştirin
  • Sosyal sağlayıcılardan bilgi alın
  • ASP.NET Web API kullanarak HTTP hizmetleri ekleyin
  • Web API güvenliğini sağlayın
  • ASP.NET SignalR ile gerçek zamanlı web ekleyin
  • Scaffolding kullanarak bileşen ekleyin
  • Browser Link ile uygulamanızı test edin
  • Projenizi paylaşın
